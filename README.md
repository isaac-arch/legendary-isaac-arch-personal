# Isaac Arch - Personal Portfolio v1

A colorful and responsive personal portfolio built using **HTML, CSS, and JavaScript**.  
Showcases sections like Home, About, Projects, and Contact with smooth animations.

##  Features
- Gradient background animation  
- Smooth scrolling navigation  
- Responsive design for all devices  
- Contact form (frontend only)  
- Eye-catching color transitions  

## Tech Stack
- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (DOM Manipulation)

## How to Use
1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Explore all sections and features
